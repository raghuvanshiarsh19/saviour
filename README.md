# 📝 Todo List Application

A modern, fully-functional todo list web application with persistent local storage. Create, manage, and organize your tasks effortlessly!

## ✨ Features

### Core Functionality
- ✅ **Add Tasks** - Quickly add new tasks using the input field or Enter key
- ✅ **Complete Tasks** - Mark tasks as complete with checkboxes
- ✅ **Delete Tasks** - Remove individual tasks from your list
- ✅ **Clear Completed** - Bulk delete all completed tasks
- ✅ **Filter Tasks** - View All, Active, or Completed tasks
- ✅ **Task Counter** - See how many active tasks remain
- ✅ **Local Storage** - All tasks persist even after closing your browser!

### Design & UX
- 🎨 Beautiful gradient purple theme
- 📱 Fully responsive (works on mobile, tablet, desktop)
- ⚡ Smooth animations and transitions
- 🎯 Intuitive user interface
- 🔒 XSS protection with HTML sanitization
- 💬 User notifications and confirmations

## 🚀 How to Use

### Getting Started
1. Clone or download this repository
2. Open `index.html` in your web browser
3. Start adding tasks!

### Adding Tasks
- Type your task in the input field
- Click "Add" button or press Enter
- Your task appears in the list

### Managing Tasks
- **Complete a task**: Click the checkbox next to the task
- **Delete a task**: Click the Delete button
- **Clear all completed**: Click "Clear Completed" button

### Filtering Tasks
- **All**: View all tasks
- **Active**: View only incomplete tasks
- **Completed**: View only completed tasks

## 💾 Local Storage

Your tasks are automatically saved to your browser's localStorage every time you:
- Add a new task
- Complete/uncomplete a task
- Delete a task

**Note**: localStorage is specific to each browser and domain. Data won't sync across different browsers or devices.
